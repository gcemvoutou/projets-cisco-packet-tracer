# 🌐 Infrastructure Réseau Cisco – Interconnexion Multi-Sites

![Cisco Packet Tracer](https://img.shields.io/badge/Cisco-Packet%20Tracer-1BA0D7?logo=cisco&logoColor=white)
![Type](https://img.shields.io/badge/Type-Scolaire-blue)

## 📋 Description

Conception et déploiement d'une infrastructure réseau complète sous **Cisco Packet Tracer**, simulant l'interconnexion de deux entreprises fictives via un cœur de réseau Cisco.

Le projet couvre l'adressage IP, le routage inter-sites, ainsi que la mise en place de services réseau critiques : DHCP, DNS et hébergement d'un site intranet.

## 🎯 Compétences démontrées

- Conception d'une topologie réseau multi-sites
- Configuration de routeurs Cisco (interfaces, routage statique)
- Déploiement d'un serveur DHCP avec plages d'adresses et réservations
- Mise en place d'un serveur DNS avec enregistrements de type A
- Hébergement d'un site intranet (HTTP) accessible par nom de domaine
- Dépannage réseau méthodique (ping étape par étape, mode simulation)
- Analyse des échanges réseau en mode simulation (DNS, ICMP)

## 🛠️ Technologies utilisées

| Outil | Rôle |
|---|---|
| Cisco Packet Tracer | Simulation réseau |
| Routeur Cisco 2911 | Interconnexion inter-sites |
| DHCP | Attribution dynamique d'adresses IP |
| DNS | Résolution de noms de domaine |
| HTTP | Hébergement du portail intranet |
